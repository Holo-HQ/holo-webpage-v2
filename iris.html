IRIS + PRISMA EDGE

La combinación de software móvil y hardware táctico que lleva la telemetría, comunicaciones seguras y reportes operacionales directamente al operador en campo.

NEXXO C2 // IRIS — FIELD OPS APP
/01 — IRIS Software

Inteligencia de Campo
en tu Dispositivo

App móvil diseñada para oficiales y operadores en el teatro de operaciones. Funciona en condiciones de baja conectividad.

  • /01Geolocalización GPS en formato MGRS (estándar OTAN) con actualización continua y reporte automático al COP
  • /02Monitoreo de combustible, municiones y estado operacional con indicadores visuales de nivel
  • /03Botón REPORTAR para SITREPs rápidos, solicitudes MEDEVAC y novedades desde campo con georreferencia
  • /04Mapa táctico embebido con posición propia y de unidades cercanas. Operación offline con sincronización posterior
NEXXO C2 // PRISMA EDGE — TACTICAL HARDWARE
/02 — PRISMA EDGE Hardware

Edge Computing
en Campo de Batalla

Dispositivo ruggedizado diseñado para las condiciones extremas del teatro de operaciones colombiano.

  • /01Microcontrolador ARM de bajo consumo con capacidad de procesamiento edge para decisiones autónomas sin conectividad
  • /02Módulo GPS de alta precisión con capacidad de reporte en formato MGRS y coordenadas decimales
  • /03Pantalla OLED/e-ink para visualización de estado, coordenadas y mensajes en condiciones de baja visibilidad
  • /04Carcasa resistente a polvo, agua y golpes. Batería de larga duración para operaciones extendidas sin recarga
/03 — Capacidades Conjuntas

Software + Hardware
Trabajo Integrado

Operación Sin Conectividad

Tanto IRIS como PRISMA EDGE funcionan completamente offline. Los datos se almacenan localmente y se sincronizan automáticamente cuando se recupera la conexión.

Reporte Automático de Posición

PRISMA EDGE transmite la posición del operador al COP (RADAR) de forma automática mediante el módulo de comunicación integrado, sin intervención del usuario.

Comunicación Segura

Canal cifrado entre el dispositivo PRISMA y el servidor NEXXO C2. Protocolos resilientes que adaptan el ancho de banda a las condiciones de la red disponible.

Diseñado para Colombia

Probado en las condiciones del teatro colombiano: selva, montaña, ríos y costa. Resistencia a temperaturas extremas, humedad y movimiento constante.
